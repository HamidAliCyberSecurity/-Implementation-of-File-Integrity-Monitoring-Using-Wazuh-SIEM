# File Integrity Monitoring Using Wazuh SIEM

## Overview
This project implements Wazuh SIEM for File Integrity Monitoring (FIM) to monitor critical system files on both Linux and Windows endpoints. It demonstrates centralized security monitoring, real-time alerting, and threat detection using custom FIM rules for SOC operations.

## Key Features
- Monitors critical system files for unauthorized creation, modification, or deletion.  
- Centralized management using Wazuh Manager with Linux and Windows agents.  
- Custom FIM rules for real-time threat detection and alerts.  
- Supports SOC workflows and incident response analysis.

## Tools & Technologies
- Wazuh SIEM  
- File Integrity Monitoring (FIM)  
- Kali Linux & Windows Agents  
- Security Monitoring & SOC Operations
